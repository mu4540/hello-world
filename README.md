# 寒霜獵人 ❄️ Frost Hunter

一個原創的 HTML5 雪原狩獵小遊戲：走出營地、獵捕冰原巨獸、撿肉回營升級裝備與招募夥伴。

**🎮 立即遊玩：https://mu4540.github.io/hello-world/**

手機（iPhone/Android）直接用瀏覽器開啟即可；用 Safari「分享 → 加入主畫面」可全螢幕遊玩。

## 操作

- 🕹️ 拖曳螢幕（或鍵盤 WASD / 方向鍵）移動
- 🔨 靠近野獸自動攻擊
- 🥩 撿起獵物掉落的肉，站在營地的虛線圈上消耗肉升級
- 🏆 擊敗最北方的「冰原巨獸」獲勝
